7/29/2022 - Release Notes

Features Added


Mosaic PowerSwitch Zero loans - Mosaic’s PowerSwitch Zero are now available in Sales Mode.

Custom Utility Rates- Added ability to create and customize utility rates in New Aurora

AutoDesigner API: Minimum TSRF / SAP in Energy Mode-   API users to may now set minimum Total Solar Resource Fraction (TSRF) or Solar Access Percentage (SAP) in Energy Mode.

Lead Capture AI azimuth configuration - Users may now define the range of roof plan azimuths that are eligible for panel placement.

New Webhook: Project created - A new Project Created webhook enables developers to build workflows that notify systems in real time when projects are created in Aurora, whether from within the app or via API.
