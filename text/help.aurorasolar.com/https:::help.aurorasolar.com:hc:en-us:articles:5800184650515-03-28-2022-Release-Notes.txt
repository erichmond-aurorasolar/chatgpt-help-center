03/28/2022 Release Notes

Features Added


Design Mode now includes the ability to adjust NEM rules on the fly, including Export Rate, True-Up period and Net Surplus Compensation information.

Features Updated

Corrected an issue where in some cases CAD could become unstable and crash if an inverter is connected to a disconnect. 
Restricted ability to resubmit a design request to designs which were completed in the last 30 days.
Corrected an issue where deleting a financing option that is used in a Sales Mode proposal would cause the financing page to display an error.
