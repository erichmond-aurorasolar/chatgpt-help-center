4/1/2021 Release Notes

Features Added

Added more descriptive verbiage for 3D Modeling service rejection reasons.
Added new API endpoint for entering consumption data.

Features Updated

Updated verbiage for 3D Modeling service rejection reasons to be more descriptive.
Corrected an issue that caused line item pricing to not update the proposal pricing when the quantity is updated. 
Corrected an issue causing the viewports on the shade report to be off-center when the original project location marker had been moved. 
Corrected an issue causing validation errors when adding a new custom utility rate outside of the US.
