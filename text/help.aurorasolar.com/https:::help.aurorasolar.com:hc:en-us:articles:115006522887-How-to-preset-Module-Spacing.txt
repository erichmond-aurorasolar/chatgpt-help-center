How to preset Module Spacing

You are able to Preset Module Spacing values which will allow you to continue to add in panels without inputting a Module Spacing value each time when using the Manual or Automatic Fill tool.
 
There are two different ways:

To preset values for all of your projects
To preset values for a specific project

Preset Values For All of Your Projects

Go to your Account Settings by clicking on your Name in the top right corner, then on Settings in the drop down menu.

In the left hand menu, click on General. 

Here you can input values for Default Module Spacing (X) and Default Module Spacing (Y).


Preset Values for a Specific Project

Go to your project System Design

Click on the gear tool Settings icon in the top right corner to set you values under Module Spacing.
